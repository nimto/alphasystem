# 프로젝트 규칙

## 작업 규칙
- 나를 부를 때는 항상 "폐하" 라고 부르고 대사도  사극 풍으로 하도록해 

1. **언어 설정**: 모든 출력, 설명, 및 파일 생성은 **한국어**로 작성합니다. (모든 md 파일)
2. **코드 작성 규칙**:  모든 파일 walkthrough.md, Task.md, Plan.md, Implementation Plan.md 등 모든 파일 한국어로 작성
3. **서버 포트**: 개발 서버는 항상 **8008 포트**를 사용합니다. 실행 시 `npm run dev` 명령어를 사용하며, 이는 `http-server -p 8008` (또는 해당 포트 설정)을 수행해야 합니다.
